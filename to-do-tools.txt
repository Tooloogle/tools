# Tool Backlog & Removal Log

This file tracks tools that were removed from the codebase and the reason why.
Do not re-scaffold any of the names below via `npm run new` without first revisiting the rationale.

---

## Backlog — Could be implemented later (13 tools)

### [Browser] Requires complex File / Canvas APIs (5)
- favicon-generator: Requires file upload and canvas API for favicon generation from images
- gif-to-webp-converter: Requires file upload, canvas, and format conversion libraries
- image-cropper: Requires file upload, canvas API, and interactive cropping UI
- image-filter: Requires file upload and canvas API for filter effects
- qr-code-reader: Requires camera API and QR code scanning library (jsQR)

### [Server / API] Requires server-side implementation (3)
- domain-whois-lookup: WHOIS lookups require server-side WHOIS protocol
- ssl-checker: SSL certificate checking requires server-side TLS inspection
- rsa-key-generator: RSA key generation should be done server-side for security

### [Package] Requires additional npm packages (3)
- excel-to-json-converter: Requires xlsx or sheetjs library for Excel parsing
- png-to-svg-converter: Requires image tracing library (e.g. potrace)
- punycode-converter: Requires the `punycode` package for proper IDN conversion

### [API] Requires additional libraries / browser APIs (2)
- checksum-calculator: Needs CRC32 and other checksum algorithm libraries
- hash-file-verifier: Requires FileReader API and streaming for large files

---

## Removed — Complex / unsuitable for browser (9)
Deleted because they require complex libraries or algorithms not suitable for client-side implementation:

1. ascii-art-generator - Requires complex ASCII art rendering engine
2. color-blindness-simulator - Complex color vision deficiency simulation algorithms
3. color-mixer - Advanced color space mixing calculations
4. color-palette-generator - Sophisticated palette generation algorithms
5. color-shade-generator - Complex color shade/tint generation
6. graphql-query-builder - GraphQL AST construction and validation
7. json-path-finder - Requires JSONPath evaluation library (jsonpath-plus)
8. xpath-tester - Requires XPath evaluation engine
9. json-schema-validator - Requires Ajv or similar JSON Schema validator library

---

## Removed — Apr 2026 low-value / duplicate cleanup (4)

1. heif-to-jpg-converter - Flaky cross-browser HEIC decoding via heavy `heic2any` dep
2. heif-to-png-converter - Same as above
3. lorem-picsum-placeholder - Trivial wrapper around picsum.photos URL; superseded by `placeholder-image-generator`
4. speed-reading-tool - Niche RSVP reader with very low SEO / usage
